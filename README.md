# CashWise AI 💰
### AI-powered financial coach for young Africans

## 🌍 The Problem
57% of young Africans have no access to financial advice. 
Not because they don't want it — but because nobody built it for them.

## 💡 The Solution
CashWise AI is a personal financial coach that helps young Africans:
- Create personalized budget plans
- Track savings and debt
- Get AI-powered financial advice in plain language
- Monitor their Financial Health Score
- Understand African financial tools like M-Pesa and SACCOs

## 🚀 Live Demo
https://app-9xxk0xxlp24h.appmedo.com

## ⚡ Features
- AI Chat Coach powered by Featherless AI (Mistral 7B)
- Personalized onboarding flow
- Savings Tracker with progress bars
- Debt Tracker
- Expense Tracker
- Financial Health Score (0-100)
- Visual Dashboard with charts
- Persistent chat history

## 🛠️ Tech Stack
- Frontend & Backend: MeDo (No-Code Platform)
- AI Model: Mistral-7B via Featherless AI
- Database: Supabase
- Hosting: MeDo Cloud
